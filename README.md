# PortfolioOptimisation
This is an R package containing functionality which enables the user to conduct portfolio optimisation on assets from the S&P 500. 

## Installation
In R:
```R
intstall.packages('devtools')
devtools::install_github("https://github.com/edwarddavis1/PortfolioOptimisation")
```
